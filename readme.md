Перед запуском
================

**bower i** - устанавливаем либы

**npm i** - устанавливаем зависимости

Если не установлен gulp, то 
**npm i -g gulp**

Если не установлен bower, то **npm i -g bower**


Запуск
================

Для запуска команда в консоли  
**gulp**  

или открываем в папке app index.html
